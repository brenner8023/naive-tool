# naive-tool
> 基于Vue3、TypeScript实现的Chrome插件（扩展）

技术栈：
- Vue3.x
- TypeScript
- Vite
- NaiveUI

功能：
- [x] Interceptor(请求拦截修改器)
- [x] Json2TS
- [x] JsonDiff

### 测试
- xhr请求可以访问简书进行测试
- fetch请求可以访问知乎进行测试
